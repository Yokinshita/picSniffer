本程序所需环境：
Windows 10
Visual Studio 2017

Python部分：
Python 3.6（包含Tkinter）
Pillow 包
opencv-contrib-python==3.4.1.15 包
numpy 包
tensorflow 包

其他：
Inception-v3模型——https://github.com/taey16/tf/tree/master/imagenet（要在代码里写好引用路径）
Tesseract-OCR——文字识别部分需要用到的库


使用Visual Studio 2017打开“毕业设计-嗅图狗.sln”，然后设置FP.py为起始运行文件即可运行。
PS：Z打头的两个.py文件是两个版本，
其中Z_FP_XiuTu-dog-version.py是单个就能运行的，Z_Frontpage.py是最初的简陋版本。这两个文件不是FP所需的。
